# Python-for-JDL-West
Python-based utility tools for warehouse use

| 序号 | 小程序名称 | 主要功能 | 访问地址 |
|------|------------|----------|----------|
| 1 | 前一日上架盘点表 | 根据前一日的上架结果和参与上架的人数，随机分配盘点储位 | [访问链接](https://python-for-jdl-west-inboundcc.streamlit.app/) |
| 2 | 动销等级计算 | 通过拣货数量和库存的比例，给货物赋予 Band 级别 | [访问链接](https://python-for-jdl-west-bandlevel.streamlit.app/) |
| 3 | 储位利用率 | 识别使用率为 0% 的储位，可用于入库上架和库存盘点 | [访问链接](https://python-for-jdl-west-storageusage.streamlit.app/) |
| 4 | 储位合并表 | 识别可以合并的储位，但暂时无法给出合并储位建议 | [访问链接](https://python-for-jdl-west-storagecombine.streamlit.app/) |
| 5 | Upph by Location | 计算触碰储位，为小件型拣货得出更合理的 UPPH | [访问链接](https://python-for-jdl-west-locationupph.streamlit.app/) |
